A fast, fully static portfolio built with Astro featuring a dark/light theme toggle. Includes a homepage, CV page, paginated project listings, and dynamic project detail pages generated from a projects collection. Pages are pre-rendered using getStaticPaths, with GitHub Actions automating build and deployment to GitHub Pages.

This is the website you are currently viewing.

Astro is a modern framework for building fast, content-focused websites.    Astro lets you build static sites (pre-rendered HTML) that load super fast.
